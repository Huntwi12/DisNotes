# Discord Notes Pro

A Discord-inspired note-taking application designed for high-performance productivity. Features include project-based organization, page-level notes, a focus dashboard, and local CSV data persistence.

## Getting Started

1. **Install Dependencies**
   ```bash
   npm install
   ```

2. **Run the Development Server**
   ```bash
   npm run dev
   ```

3. **Launch the App**
   Open [http://localhost:3500](http://localhost:3500) in your browser.

---
*Note: This app uses a local `data/` folder for storage. Ensure this directory exists before running.*
